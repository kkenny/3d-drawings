# Summary

<p align="center">
<img src="https://github.com/mattdibi/redox-keyboard/raw/master/img/redox-logo.png" alt="Redox logo" width="400"/>
</p>

*REduced ergoDOX custom mechanical keyboard.*

This project is a case for a custom ergonomic mechanical keyboard. The plate is compatible with Cherry MX and clones switches. This entire project is supposed to be a hand-wiring project. I uploaded the schematic and an image containing the switch location to help in the wiring process. If you follow that schematic you can use the [QMK Redox firmware](https://github.com/qmk/qmk_firmware/tree/master/keyboards/redox).

**Update**: A new version of this project is available. [The Redox rev.1](https://www.thingiverse.com/thing:2886662) support a custom PCB (designed by me =D ) to simplify the building process. More information about the project [in the Redox repository](https://github.com/mattdibi/redox-keyboard).

**Update2**: Added a modified version of the plate in which the layout uses less 1.25u keycaps in favour of 1u keycaps. The new plate files are: `PlateL1u.STL` and `PlateR1u.STL`.

Materials needed:

- **70**x: Cherry compatible switches
- **70**x: Diodes (I used 1N4148)
- **70**x: Cherry compatible keycaps
  - **10**x (**8**x for 1u Layout): 1.25u keycaps
  - **6**x: 1.5u keycaps
  - **54**x (**56**x for 1u Layout): 1u keycaps
- **14**x: M3 8mm screws
- **2**x: 4 poles 3.5 mm plug
- **2**x: Arduino pro micros
- **2**x: USB micro male to female cable/adapter
- A lot of cable
- A lot of soldering
